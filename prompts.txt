2.promt given to setup authentication :
Goal: Add email/password authentication with JWT in HttpOnly cookie for better UX.

Backend:
- New User model: { name, email(unique, lowercase), passwordHash }
- New routes (mounted /api/auth):
  POST /register {name,email,password} -> set HttpOnly 'token' cookie, return user{id,name,email}
  POST /login {email,password} -> set cookie, return user
  GET /me -> requires auth, return {user}
  POST /logout -> clear cookie
- requireAuth middleware (verifies JWT from cookie, set req.user)
- .env additions: JWT_SECRET, JWT_EXPIRES=7d, COOKIE_SECURE=false
- CORS: origin http://localhost:5173, credentials true
- cookie-parser added

Frontend:
- axios instance with withCredentials: true
- Zustand auth store: { user, loading, error, hydrate(), login(), register(), logout() }
- Pages: Login.jsx, Register.jsx (react-hook-form)
- Navbar: if user -> "Hi, {firstName}" + Logout; else Login/Register
- Auto-hydrate on app load (calls /api/auth/me)

Acceptance:
- Register -> auto-login cookie set; refresh keeps user
- Login/Logout works; /api/auth/me returns 401 when signed out
